El proyecto banco solar consiste en la creación del sistema de transferencias del Banco Solar.

Se pueden ingresar nuevos usuarios, con un balance inicial. A partir de este balance se pueden realizar transferencias de saldos entre los usuarios.

  • Creación y conexión a una base de datos PostgreSQL con Node.
  • Realizar consultas y modificaciones a los valores de esta con Node y el paquete pq.
  • Manejo de Errores con códigos de estado HTTP
